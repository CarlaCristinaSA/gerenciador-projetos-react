### O que é React?

* É uma biblioteca JS para criação de interfaces
* Utilizado para construir SPAs (Single Page Aplication)
* Baseado em componentes
* Utiliza JSX para renderizar HTML
* Aplica o Virtual DOM para realizar as alterações de DOM
* Podemos adicionar a um projeto ou criar um projeto com ele

### Instalação
* Para instalar o React vamos utilizar a ferramenta chamada Create React App
* Recebemos todos os arquivos da biblioteca e temos como executá-la
* Para utilizar precisamos do node e do npm
* Esta ferramenta também otimiza o app gerado para produção
* É possivel iniciar a aplicação com npm start

### Create React App
npx create-react-app meu-projeto
cd meu-projeto
npm start 
  Local:            http://localhost:3000
  On Your Network:  http://192.168.0.7:3000
