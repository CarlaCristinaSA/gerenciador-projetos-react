## Sobre o React

O React é uma biblioteca JavaScript de código aberto focada na construção de interfaces de usuário (UI).

* **Componentização:** Baseado em pequenos blocos de código reutilizáveis e independentes.
* **JSX (JavaScript XML):** Extensão de sintaxe que permite escrever HTML dentro do JavaScript.
* **Virtual DOM:** Otimiza a renderização, atualizando apenas o que é necessário no DOM real, garantindo alta performance.
* **Ecossistema SPA:** Ideal para criar aplicações de página única, proporcionando uma experiência de navegação fluida.

## Instalação e Inicialização

Neste projeto, utilizamos a ferramenta **Create React App** para configurar o ambiente de desenvolvimento de forma otimizada.

### 1. Criando o projeto
Para iniciar um novo projeto do zero, utilize o comando:
```bash
npx create-react-app meu-projeto
```

### 2. Acessando a pasta
```bash
cd meu-projeto
```

### 3. Rodando a aplicação
Para iniciar o servidor de desenvolvimento:
```bash
npm start
```

Após o comando, a aplicação estará disponível em:
* **Local:** `http://localhost:3000`
* **Rede:** `http://192.168.0.7:3000`
